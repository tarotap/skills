# Tarotap Agent Skills

Skills that give AI agents a complete, practiced tarot reading craft — real
randomness, public-domain Rider–Waite–Smith card imagery, and
position-by-position interpretation.

## Install

```bash
npx skills add tarotap/skills
```

Works with Claude Code, Cursor, Codex, Copilot, Windsurf, and 40+ other
agents via the open agent skills ecosystem.

## Skills

| Skill | Description |
| --- | --- |
| `tarot-reading` | Interactive three-round tarot reading: frame the question, pick a spread (9 layouts up to the full Celtic Cross), let the user choose cards from the shuffled deck, draw with cryptographic randomness and fair reversals, then interpret each position with card imagery. |

## About

Built by Tarotap, an AI tarot reading platform serving readings in 21
languages.

https://tarotap.com/en

Card imagery: 1909 Rider–Waite–Smith deck, public domain, served from
Wikimedia Commons.
